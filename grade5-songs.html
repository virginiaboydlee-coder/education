📝 How to update this song database (teachers only)

Adding / editing songs: Find the songsData array in the <script> section. Each song uses this format:

{ artist: "Artist Name", title: "Song Title", chordsRaw: "C G Am F", listenUrl: "filename.mp4", chordUrl: "filename.pdf" }
  • chordsRaw – space-separated chords (e.g. "C F Am G")
  • listenUrl – exact filename of your .mp3/.mp4 (or "" if none)
  • chordUrl – exact filename of your chord sheet PDF (or "")

Difficulty is automatic — 1–3 chords = Easy, 4–5 = Medium, 6+ = Hard.

File location: Place media files in the same folder as this HTML file, or update the paths accordingly.
